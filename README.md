# Matki pszczele Subik x Clifford — v11

Wersja z synchronizacją Supabase, autoodświeżaniem i przygotowanym Web Push.

Frontend:
- wrzuć pliki z tego folderu na GitHub/Vercel,
- otwórz aplikację na iPhonie z linku HTTPS,
- dodaj do ekranu początkowego,
- uruchom z ikonki i kliknij „Włącz powiadomienia”.

Backend powiadomień:
- pliki SQL i Edge Function są w folderze `supabase/`.
